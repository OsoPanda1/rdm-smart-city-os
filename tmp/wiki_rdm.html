Purpose and Scope

Real del Monte Explorer (RDM Digital) is a full-stack digital platform for the Pueblo Mágico of Real del Monte, Hidalgo. It serves as a comprehensive tourism guide, a community hub, and a business directory, integrating advanced features like the REALITO AI assistant powered by the ISABELLA AI™ v4.0 Enterprise engine. The platform aims to preserve local culture while providing modern digital services for tourists, local businesses, and administrators.

For detailed implementation information:

  • Frontend structure and routing: see Application Root & Routing
  • Backend API and database models: see Backend Services
  • Data fetching patterns: see Data Flow Architecture
  • User authentication and permissions: see User Roles & Permissions

Core System Capabilities

1. Realito AI Assistant (ISABELLA AI™ v4.0)

The platform features a sophisticated AI assistant named Realito, which utilizes the TAMV (Transformer-based Autonomous Memory Vector) architecture. It provides contextual information about Real del Monte through a 12-layer transformer model and a "Neural Memory Mesh" for episodic memory.

Key AI Components:

  • TAMV Core Processor: Handles tokenization and inference src/features/ai/isabellaTamvBase.ts#43-46
  • Neural Memory Mesh: Provides semantic search and temporal consolidation src/features/ai/isabellaTamvBase.ts#48-51
  • Audit System: Ensures cryptographic traceability of AI interactions src/features/ai/isabellaTamvBase.ts#59-62

Sources: docs/REALITO-ISABELLA-TAMV-ENTERPRISE.md#46-66 src/features/ai/isabellaTamvBase.ts#22-64

2. Interactive Tourism Content

The application serves rich, animated content across specialized domains:

  • Historia: Timeline-based heritage exploration src/App.tsx#50
  • Gastronomía: Deep dive into local cuisine, specifically the "Paste" culture src/App.tsx#54
  • Dichos: A collection of 47 "Dichos Mineros" (mining sayings) categorized by theme src/App.tsx#61-62
  • Rutas: Interactive tourist routes with difficulty and practical info src/App.tsx#56
